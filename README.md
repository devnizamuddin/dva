# DVA CLI

**DVA** is a CLI tool to automate your daily developer tasks.

**Latest Version: 1.0.6**

## Features
- 🗂️ Git Version Management
- 💙 Flutter Project Helper
- 📝 Note Book Manager
- 🔠 TextCase Converter
- 🍎 MacOS Task Helper

## Installation

```bash
git clone https://github.com/devnizamuddin/dva.git
cd dva
./install.sh
```
